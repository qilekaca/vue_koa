
```

```

```
npm run build
```

```
npm run lint
```

# 注释
新建一个server文件夹，并在此文件夹下新建5个文件夹
1. config // 配置文件夹
2. controllers // controller-控制器
3. models // model模型
4. routes // route-路由
5. schema // schema-数据库表结构  
在根目录创建一个app.js文件作为koa的启动文件
